1. What is an Annotation?
	
	i) A new feature from Java 5.
	ii)  kind of meta data that you can enter in your code.
	iii) They do not have any direct effect on operations of the code the annotate.

	
2. Uses of Annotaion?

	i) Information for the compiler : Annotations can be used bt the compiler to detect 
										errors or suppress warnings.
	
	ii) Compile time processing and deployment time processing : Software tools can process annotation 
																	information to generate code, XML files etc.
	
	iii) Runtime processing : Some annotations are available to be examined at runtime
								via Java Reflection.
